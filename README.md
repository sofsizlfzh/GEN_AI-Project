# 📌 Project Overview

Smart Email & Letter Generator is an AI-powered productivity application built using **Python**, **Streamlit**, and **Groq API**.

It helps students, job seekers, freelancers, and professionals create polished communication documents in seconds.

Users can generate:

- Professional Emails  
- Cover Letters  
- Job Applications  
- Complaint Letters  
- Business Proposals  
- Thank You Letters

---

# ✨ Features

## 🧠 AI Writing Engine

- Powered by **Groq API**
- Uses **Llama 3.1 8B Instant**
- Fast response generation
- Human-like professional writing

## 🎯 Smart Controls

- Multiple document types
- Tone selection
- Length selection
- Multi-language output

## 📄 Resume Upload

Upload:

- PDF Resume
- DOCX Resume
- TXT Resume

AI automatically reads your resume and uses it in the generated letter.

## 🎨 Premium UI

- Dark modern theme
- Sky-blue borders
- Responsive layout
- Clean two-column design

## 📥 Export

- Download generated document as TXT
